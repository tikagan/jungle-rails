# Jungle

A mini e-commerce application built with Rails 4.2 for purposes of teaching Rails by example.

## Dependencies

* Rails 4.2 [Rails Guide](http://guides.rubyonrails.org/v4.2/)
* PostgreSQL 9.x
* stripe
* sass-rails
* uglifier
* jquery-rails
* turbolinks
* jbuilder
* sdoc
* bcrypt
* puma
* bootstrap-sass
* font-awesome-rails
* money-rails
* carrierwave'
* rmagick

## Development Dependencies
* faker'
* byebug
* quiet_assets
* dotenv-rails
* web-console', '~> 2.0'
* spring
